# 🎬 YouTube Clone (React + YouTube API)

Projeto desenvolvido com React simulando a interface do YouTube, consumindo a API oficial do YouTube para exibição de vídeos, busca e player.

## 🚀 Funcionalidades

- 📺 Listagem de vídeos em alta (trending)
- 🔍 Busca de vídeos em tempo real
- ▶️ Player de vídeo integrado (página /watch)
- 📌 Vídeos relacionados na lateral
- 🎨 Interface inspirada no YouTube
- 👤 Sistema de login (simulado/local)

## 🛠️ Tecnologias utilizadas

- React
- TypeScript
- Styled-components
- React Router DOM
- YouTube Data API v3
